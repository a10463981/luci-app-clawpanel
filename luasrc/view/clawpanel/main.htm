📁 统一目录结构 /Configs
所有 ClawPanel 和 OpenClaw 相关文件统一存放在 /Configs 目录下，便于管理、备份和迁移。
# 选择存储盘后，目录结构如下（示例）:
# /mnt/sda1/Configs/
  ├─ clawpanel/          # ClawPanel 程序 + 数据
  │    ├─ clawpanel      # Go 二进制
  │    └─ data/clawpanel.json
  ├─ openclaw/           # OpenClaw npm 包（可选）
  ├─ .openclaw/          # OpenClaw 工作区
  └─ .openclaw-work/  # 运行时目录
📦 系统重装后：安装插件 → 运行 CP_BASE_PATH=/mnt/sda1 sh install.sh 即可恢复

首次使用：点击「安装 / 重装」→ 选择外置存储盘 → 等待完成
